# 이 문서는 프로젝트를 진행하면서 검색을 통해 정보를 공유하는 문서입니다.

### 중요
**이 문서는 병합시 문제가 무조건 발생합니다. 같은 파일에서 수정하기 때문이죠**
**따라서 Note 폴더에 이니셜_ref_docx.md로 저장 후 안에 내용을 작성해야합니다**
**main 또는 devleop에서 이 문서에 병합합니다.**

다음과 같이 작성하시면 됩니다


##### 예시
```
!!! Note/ref_docx/"자신 이름의 이니셜"_ref_docx.md 에 작성
!!! 내용과 추가할내용은 제목으로 설명 가능시 생략가능
- 제목 : git commit 메세지에 관하여
- 내용 : git commit 메세지 룰에 대해서 설명 
- 참고자료 : https://ivory-room.tistory.com/93
- 추가할내용 : 위 자료는 commit 메세지 작성시 반드시 따라줘야 합니다.
```

##### 예시2 ( 옵시디언 자료일 경우 )
- 제목 : 프로젝트 전체 내용에 대하여
- 내용 : 이번 프로젝트에 관한 내용들이 들어있는 자료입니다
- 참고자료 : 나만의 미니 가계부(지출 관리 서비스)
- 추가할내용 : 해당 문서의 " ~~~~~~~ " 부분을 ctrl+f 로 검색해서 찾으면 됩니다.
